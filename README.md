### 🏆 Featured Project

#### 🏸 [NOVA Booking](https://github.com/TruongDev24/nova-booking) — *Premium Badminton Court Booking & Management Platform*

<p align="left">
  <img src="https://img.shields.io/badge/Next.js_15-000000?style=flat-square&logo=nextdotjs&logoColor=white" alt="Next.js" />
  <img src="https://img.shields.io/badge/NestJS-E0234E?style=flat-square&logo=nestjs&logoColor=white" alt="NestJS" />
  <img src="https://img.shields.io/badge/Prisma_ORM-2D3748?style=flat-square&logo=prisma&logoColor=white" alt="Prisma" />
  <img src="https://img.shields.io/badge/Redis-DC382D?style=flat-square&logo=redis&logoColor=white" alt="Redis" />
  <img src="https://img.shields.io/badge/PostgreSQL-316192?style=flat-square&logo=postgresql&logoColor=white" alt="PostgreSQL" />
  <img src="https://img.shields.io/badge/Socket.io-010101?style=flat-square&logo=socketdotio&logoColor=white" alt="Socket.io" />
</p>

A production-grade, service-oriented web application designed to optimize court reservations for managers and sports players. The platform guarantees concurrency control, instant booking synchronization, and automated transaction settlement.

- **⚡ Real-time Synchronization:** Leverages **Socket.io** to synchronize booking slots and maintenance flags instantly across all active clients without reloading the page.
- **🔒 Double-Booking Prevention:** Implemented an **Atomic Redis Lock** (10-minute expiry) to temporarily reserve court slots during checkout, ensuring zero double-bookings.
- **💸 Automated Payments & Refunds:** Integrated **PayOS (VietQR)** with webhook handlers for automated instant invoice confirmations and built an administrator control panel to manage automated refunds.
- **📊 Admin Analytics Dashboard:** Rich data visualization showing key performance indicators: overall revenue, occupancy rates, user retention, and court utilization hours.
- **🛠️ Service Operations:** Complete CRUD for court settings (utilizing **Cloudinary** for image assets), flexible price-by-hour policies, and full maintenance scheduling that auto-notifies impacted clients via email.
